\# OEE Dashboard and Production Defect Analysis System



A web-based application to calculate Overall Equipment Effectiveness (OEE) and analyze production performance in manufacturing industries.



\## Technologies Used

\- Python

\- Flask

\- SQLite

\- HTML



\## Features

\- Calculate Availability, Performance, and Quality

\- Compute Overall OEE

\- Store production data in SQLite database

\- Display results in a user-friendly interface



\## Formula

OEE = Availability × Performance × Quality
