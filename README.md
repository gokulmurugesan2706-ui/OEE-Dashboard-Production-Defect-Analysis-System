\# OEE Dashboard and Production Defect Analysis System



A web-based application to calculate Overall Equipment Effectiveness (OEE) and analyze production defects in manufacturing industries.



\## Features



\- Calculate Availability, Performance, and Quality

\- Compute Overall Equipment Effectiveness (OEE)

\- Analyze production defects such as:

&#x20; - Scratch Defects

&#x20; - Crack Defects

&#x20; - Dimension Errors

&#x20; - Porosity Defects

\- Identify the most frequent defect type

\- Calculate top defect contribution percentage

\- Store production and defect data in SQLite database



\## Technologies Used



\- Python

\- Flask

\- SQLite

\- HTML

\- Git and GitHub



\## OEE Formula



OEE = Availability × Performance × Quality



Where:

\- Availability = Operating Time / Planned Production Time

\- Performance = (Ideal Cycle Time × Total Count) / Operating Time

\- Quality = Good Count / Total Count



\## Project Objective



To develop a manufacturing performance dashboard that helps evaluate equipment effectiveness and identify major production defects for quality improvement.
